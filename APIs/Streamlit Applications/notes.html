Problems Faced and Solutions

1. Streamlit Not Installed

Problem: Running streamlit run app.py resulted in "Command not found".

Solution: Installed Streamlit using pip install streamlit.

2. Multipage App Not Showing Pages

Problem: The additional pages in the pages folder were not appearing in the sidebar.

Solution: Ensured that page files were correctly named and placed inside a folder named pages in the root directory.

3. Session State Not Persisting

Problem: Counter value reset on each button click.

Solution: Used st.session_state to store counter values persistently.

4. Authentication Issues

Problem: The login form did not restrict access properly.

Solution: Used a function to check authentication status and hide content accordingly.

5. Matplotlib Graph Not Displaying

Problem: Streamlit displayed an error when rendering Matplotlib plots.

Solution: Used st.pyplot(fig) instead of plt.show().

6. Pandas DataFrame Not Displaying Properly

Problem: The table appeared as plain text instead of an interactive table.

Solution: Used st.dataframe(df) instead of st.write(df) for better interactivity.

7. Plotly Graph Not Rendering

Problem: The Plotly scatter plot was not appearing.

Solution: Used st.plotly_chart(fig) to render the graph correctly.
