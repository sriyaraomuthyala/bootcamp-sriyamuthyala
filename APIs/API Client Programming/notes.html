API Client Programming - Problems & Solutions

Problem 1: SpaceX API Connection Error

Error: "Failed to resolve 'api.spacex.land' ([Errno 11001] getaddrinfo failed)".

Solution: Checked internet connection, tried alternative API endpoint https://spacex-production.up.railway.app/graphql/, and switched DNS to Google's (8.8.8.8).

Problem 2: GitHub Contributions Not Found

Scraping GitHub profile page failed due to dynamic content loading.

Solution: Used GitHub's GraphQL API instead to fetch yearly contributions.

Problem 3: Bluesky API Authentication

Error: "Invalid credentials" when posting a message.

Solution: Used an App Password instead of the normal password from Bluesky settings.

Problem 4: Fetching Exchange Rates

Hardcoded target currency instead of allowing user input.

Solution: Modified script to take currency as a command-line argument.

Problem 5: AniList API Not Fetching Descriptions

Anime descriptions contained HTML tags.

Solution: Removed HTML tags (<br>, <i>) from descriptions for clean output.
