FastAPI Development Notes

1. Virtual Environment Issues

Problem: FastAPI and dependencies were not recognized after installation.

Solution: Activated the virtual environment using source venv/bin/activate (Windows: venv\Scripts\activate).

2. Uvicorn Not Found

Problem: Running uvicorn main:app --reload resulted in a "command not found" error.

Solution: Ensured installation using pip install uvicorn.

3. CORS Issues When Testing from Frontend

Problem: API requests from frontend were blocked due to CORS policy.

Solution: Installed and enabled CORS middleware using:

pip install fastapi[all]
from fastapi.middleware.cors import CORSMiddleware
app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_credentials=True, allow_methods=["*"], allow_headers=["*"])

4. SQLite Database Not Creating Tables

Problem: After integrating SQLite, tables were not being created.

Solution: Ran Base.metadata.create_all(bind=engine) after defining models.

5. Background Task Not Executing

Problem: Background tasks were not running as expected.

Solution: Ensured the task was added properly using background_tasks.add_task(task_function, args) and verified logs.
